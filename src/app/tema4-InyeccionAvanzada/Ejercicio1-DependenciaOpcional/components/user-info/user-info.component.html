Crea un componente llamado UserInfoComponent que reciba dos dependencias mediante inyección de dependencias: AuthService y LoggerService.

Sin embargo, LoggerService no siempre estará disponible, por lo que deberás hacer uso del decorador ' Optional()' para que el componente funcione correctamente incluso si LoggerService no se proporciona.
